Ejercicios

Ejercicios de Condiciones.

1)      Ejercicio de par o impar con if-else:

a.       Solicitar al usuario un número.

b.       Comprobar si el número es par o impar utilizando una estructura condicional if-else.

c.       Mostrar un mensaje indicando si el número es par o impar.

2)      Ejercicio de calculadora básica con switch:

a.       Pedir al usuario que ingrese dos números y una operación (suma, resta, multiplicación, división).

b.       Utilizar un switch para realizar la operación seleccionada y mostrar el resultado.

3)      Ejercicio de determinar el día de la semana con switch:

a.       Pedir al usuario que ingrese un número del 1 al 7.

b.       Utilizar un switch para mostrar el día de la semana correspondiente al número ingresado.

4)      Ejercicio de determinar si un número es positivo, negativo o cero con if-else:

a.       Solicitar al usuario un número.

b.       Utilizar una estructura condicional if-else para determinar si el número es positivo, negativo o cero.

c.       Mostrar el resultado.

5)      Ejercicio de cálculo de edad con if-else:

a.       Pedir al usuario que ingrese su año de nacimiento.

b.       Calcular la edad del usuario y determinar si es mayor de edad o menor de edad utilizando una estructura condicional if-else.

6)      Ejercicio de determinar el mayor de tres números con if-else:

a.       Solicitar al usuario que ingrese tres números.

b.       Utilizar una estructura condicional if-else para encontrar y mostrar el número mayor.

7)      Ejercicio de verificación de contraseña con if-else:

a.       Pedir al usuario que ingrese una contraseña.

b.       Verificar si la contraseña cumple con ciertos criterios (longitud mínima, presencia de números, etc.) utilizando una estructura condicional if-else.

8)      Ejercicio de clasificación de triángulos con if-else:

a.       Solicitar al usuario que ingrese las longitudes de los lados de un triángulo.

b.       Utilizar una estructura condicional if-else para determinar si el triángulo es equilátero, isósceles o escaleno.

9)      Ejercicio de determinar el horario con switch:

a.       Pedir al usuario que ingrese una hora del día en formato de 24 horas.

b.       Utilizar un switch para mostrar si es mañana, tarde, noche o madrugada.

10)   Ejercicio de cálculo de descuento con if-else:

a.       Solicitar al usuario que ingrese el monto de una compra.

b.       Aplicar un descuento del 10% si la compra supera cierto valor utilizando una estructura condicional if-else.

c.       Mostrar el monto final a pagar.

Ejercicios de Ciclos de Repetición.

1) Leer 10 números e imprimir solamente los números positivos

2) Leer 20 números e imprimir cuantos son positivos, cuantos negativos y cuantos neutros.

3) Leer 15 números negativos y convertirlos a positivos e imprimir la suma de dichos números.

4) Suponga que se tiene un conjunto de calificaciones de un grupo de 40 alumnos. Realizar un algoritmo para calcular la calificación media y la calificación más baja de todo el grupo.

5) Calcular e imprimir la tabla de multiplicar de un número cualquiera. Imprimir el multiplicando, el multiplicador y el producto.

6) Simular el comportamiento de un reloj digital, imprimiendo la hora, minutos y segundos de un día desde las 0:00:00 horas hasta las 23:59:59 horas.

7) Obtener el promedio de calificaciones de un grupo de n alumnos.

8) Calcular el promedio de edades de hombres, mujeres y de todo un grupo de alumnos.

9) Encontrar el menor valor de un conjunto de n números dados. 13) Encontrar el mayor valor de un conjunto de n números dados.

10) Leer 50 calificaciones de un grupo de alumnos. Calcule y escriba el porcentaje de reprobados. Tomando en cuenta que la calificación mínima aprobatoria es de 70.
